# Samsung-Galaxy-j5-Performance-Optimization
## My experience using custom ROMs to make an old phone smooth again

I have always loved playing around with phones and trying to make weak devices work better. I started these attempts when I was around 13 years old because I had an old Samsung Galaxy J5 with only 1.5GB of RAM and it was very slow. My main goal was simple I just wanted to make the device smooth enough for daily use and Performance Optimization for Mobile Gaming.

To find the best option I spent a lot of time searching and digging through the XDA Developers forums. Through this process I ended up testing and flashing more than 20 different custom ROMs to compare their speed and how they handle the memory. After a lot of trials I found out that Bootleggers OS and MSM Xtended were the absolute best in making the device fast and stable.

Inside Bootleggers OS there were specific settings that helped me make the system much lighter. I managed to turn off all the system animations completely to stop the lag and stop the CPU from getting heavy. I also killed the background services that were eating up the small RAM and changed the hardware buttons settings so they respond faster.

The other good option was MSM Xtended which was amazing at handling the cache and keeping the system smooth without crashing. 

This long experience showed me that you do not need an expensive phone to understand how a system works. Dealing with these limits since I was 13 taught me how to find solutions by myself and make any weak hardware work perfectly.
